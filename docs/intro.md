# Pepper❤️Craft 服务器官方文档
> Written by: AtomicalPepper

👋 欢迎来到我们的服务器文档！

  本文档的**大部分内容**只适用于PepperCraft服务器, 请各位在游玩其他服务器或原版生存的时候不要过度依赖~

理解本文档的内容时, 你应该了解: 
- `<xxx>` 表示参数。
  
  例如: 命令`/tpa <玩家名>`意味着你需要将`<玩家名>`替换为一个玩家的游戏名。
  
  假设你需要对名为Pepper的玩家执行命令:
   
  **/tpa Pepper** ✓
  
  */tpa \<Pepper\>* ×
  
  */tpa \<玩家名\>* ×
