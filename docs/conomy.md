# 经济系统
PepperCraft服务器采用**钻石**作为通用货币。

当然，物品状态的钻石不能用作交易货币，你需要先将钻石存入银行。

## 钻石银行系统
### GUI界面:
- 命令: ```/bank gui```
- 格式: ```/bank gui```
- 功能: 打开GUI管理页面。你可以通过按钮存储钻石、取出钻石。
- 图片: 懒得截图了...
### 存入钻石
- 命令: ```/bank save```
- 格式: ```/bank save <数额>```
- 功能: 将背包中一定数量的钻石存入银行。
